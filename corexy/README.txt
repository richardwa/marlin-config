- 3d touch
- no heated bed

# GCODE cheat sheet
G211 S0 - turn off software endstops
G211 S1 - turn on software endstops
G1 - move X,Y,X,E
G28 - Home
G29 - level bed
M420 V - show leveling data

M503 - settings report
M500 - save settings
